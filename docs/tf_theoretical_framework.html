Theoritical Framework

4층 분석 프레임워크

본 연구의 개념적 프레임워크는 4개의 분석 층위로 구성된다.

층위 구성요소 설명
이론층 8개 이론적 기반 문화산업론, 창조산업론, 가치사슬론, 플랫폼경제론, 장이론, 혁신확산이론, 문화경제학, 산업조직론
구성개념층 핵심 구성개념 산업 정체성, 기술-미디어 융합, 글로벌 유통, 정책 거버넌스
변수층 IV/DV/Med/Mod 변수 체계 독립변수 4개, 종속변수 3개, 매개변수 2개, 조절변수 2개
방법론층 분석 방법 PRISMA 2020, 키워드 네트워크, STM, 교차비교 갭 분석

각 층위는 상위 층위로부터 이론적 정당성을 부여받으며, 하위 층위로 조작화(operationalization)된다.

변수 체계

독립변수 (Independent Variables)

변수 조작적 정의 빈도
기술혁신 VR/AR, AI, 메타버스, 블록체인 등 신기술의 콘텐츠 산업 도입 정도 326편 (62%)
정책지원 정부의 재정 지원, 세제 혜택, 규제 완화 등 산업 진흥 정책의 강도 88편 (17%)
글로벌화 콘텐츠의 해외 수출, 국제 공동제작, 문화적 할인 극복 53편 (10%)
플랫폼 생태계 OTT, 소셜미디어, 앱마켓 등 디지털 플랫폼의 시장 지배력 43편 (8%)

종속변수 (Dependent Variables)

변수 조작적 정의 측정 지표
산업 성장 콘텐츠 산업의 양적 성장 매출액 증가율, 수출액, GDP 기여도, 고용자 수
경쟁력 국내외 시장에서의 경쟁적 지위 국제경쟁력 지수, LQ지수, 시장점유율
부가가치 생산 과정에서 창출되는 경제적-문화적 가치 부가가치율, 산업연관 파급효과

매개변수 (Mediating Variables)

변수 매개 경로
콘텐츠 품질 기술혁신 -> 콘텐츠 품질 -> 경쟁력
인력 역량 정책지원 -> 인력 역량 -> 부가가치

조절변수 (Moderating Variables)

변수 조절 효과
규제 환경 기술혁신 x 규제환경 -> 산업 성장
시장 구조 플랫폼 생태계 x 시장구조 -> 경쟁력

변수 구조도

[독립변수]              [매개변수]           [종속변수]

기술혁신 (326편) -----> 콘텐츠 품질 -------> 산업 성장
정책지원 (88편) ------> 인력 역량 ---------> 부가가치
글로벌화 (53편) ---------------------------> 경쟁력
플랫폼 생태계 (43편) ---------------------> 부가가치

[조절변수]
규제 환경 (50+88편) ------(조절)---------->
시장 구조 (99편) ---------(조절)---------->

Hypothesis

본 연구는 가설 검증형 연구가 아닌 체계적 문헌고찰(SLR)이므로, 전통적 가설 대신 4개의 연구질문(RQ1-RQ4)을 통해 콘텐츠 산업 연구의 지적 지형을 탐색적으로 분석한다. 위의 변수 체계는 524편의 문헌에서 귀납적으로 도출된 것이며, 향후 실증 연구에서 인과 관계를 검증하기 위한 분석적 준거로 제안된다.
