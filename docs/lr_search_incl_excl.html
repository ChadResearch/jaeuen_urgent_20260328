Searching and Inclusion & Exclusion

검색 전략

Round 1: KCI 검색

  • 데이터베이스: KCI (한국학술지인용색인)
  • 검색어: 제목에 “콘텐츠” AND “산업”
  • 결과: 416건 (2004-2026)

Round 2: WoS 검색

  • 데이터베이스: Web of Science
  • 검색어: TS=(“content industry” OR “creative industry” OR “cultural industry”)
  • 결과: 489건 (1996-2026)
  • 정제: 주제 무관 178건 제거 -> 311건

합산

  • 총 수집: 905건 (KCI 416 + WoS 489)
  • 중복 제거: -10건 (DOI/제목 매칭) -> 895건

PRISMA 2020 스크리닝

스크리닝 도구

  • ScholarRAG v1.2.4 (systematic_review 모드)
  • ChromaDB + all-MiniLM-L6-v2 (title + abstract 임베딩)
  • 적합성 점수(relevance score) 기반 자동 분류

스크리닝 결과

분류 건수 비율 기준
자동 포함 449건 50.2% score >= 40
검토 대기 250건 27.9% 40 > score > 하위 경계
자동 제외 196건 21.9% score <= 하위 경계

포함/제외 기준

포함 기준 (Inclusion Criteria):

  • 콘텐츠 산업을 주요 연구 대상으로 다루는 논문
  • 문화산업, 창조산업, 디지털 콘텐츠 산업 관련 연구
  • 콘텐츠 비즈니스, 정책, 기술혁신, 노동 등 산업적 관점의 연구
  • KCI 등재지 또는 WoS 인덱싱 학술지에 게재된 논문

제외 기준 (Exclusion Criteria):

  • 콘텐츠 산업과 관련 없는 자연과학, 의학, 공학 논문
  • ‘콘텐츠’ 또는 ’산업’이 다른 맥락으로 사용된 논문
  • 학술대회 발표자료, 서평, 편집자 논평 등 비학술논문

최종 선정

출처 선정 비율
KCI 315편 60.1%
WoS 209편 39.9%
합계 524편 100%

최종 524편 = 자동포함 449건 + 검토대기 상위 30% (75건)

적합성 점수 분포

범위 표시 설명
score >= 75 [+] 상위 10% – 핵심 문헌
35 < score < 75 [=] 중간 80%
score <= 35 [-] 하위 10%
